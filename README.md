# noob-first-game-project
这是一个刚开始学Unity的菜鸟根据教程制作的第一个Unity开发项目
同时也是一个刚弄清楚GitHub以及Git的菜鸟上传的第一个项目
该项目并没有什么高深的技巧和新奇的创意，只是一个菜鸟Unity学习者的第一个项目记录。
首先要感谢b站良心Up主 秦无邪OvO 所提供的开源项目和视频教学，能够让我这个自学C#基础，并且还不能完全熟练运用C#来进行项目制作的学习者能够制作自己的第一个看上去算是游戏的项目。这是他bilibili空间的地址https://space.bilibili.com/335835274?spm_id_from=333.788.upinfo.head.click 
另外，该项目的所有素材都由秦无邪OvO制作，如果有需要可以跳转这个链接去获取https://github.com/zs8861/2D-Platform
这是我第一次制作一款可以动，有动画，还有一些基本功能的小游戏。虽然脚本写的非常糟糕，文件管理也并不规范，但是我十分开心自己能够完成这个项目并且学习了很多关于C#和Unity的基础知识。为了成为一个真正的开发者，我会在这一步基础上更加努力。https://docs.github.com/github/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax
